## What is Ascend?

**Ascend** is an open-source Experimentation platform that empowers teams to make data-driven decisions through controlled experiments. It provides a complete solution for running experiments across diff mobile applications with minimal integration effort.
It also provides support to create and manage audience(cohorts) using both static(ID-based) and dynamic(Query based) rules over data sources.
