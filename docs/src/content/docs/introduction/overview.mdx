## What is Ascend?

**Ascend** is an open-source A/B testing Experimentation platform that empowers teams to make data-driven decisions through controlled experiments. It provides a complete solution for running experiments across different mobile applications with minimal integration effort.

It also provides support to create and manage audience(cohorts) using static(ID-based) .
